# 🚀 Mes projets JavaScript

Projets réalisés pour apprendre le JavaScript (HTML / CSS / JS vanilla).
Chaque projet m'a permis de pratiquer de nouveaux concepts du langage.

## 📂 Projets

### 1. 🚌 Passenger Counter (`01-passenger-counter`)
Compteur de passagers avec sauvegarde des totaux.
**Concepts :** variables, fonctions, DOM, événements, concaténation.

### 2. 🏀 Basketball Scoreboard (`03-basketball-scoreboard`)
Tableau de score pour deux équipes (+1 / +2 / +3, reset).
**Concepts :** paramètres de fonction, factorisation, fonctions fléchées.

### 3. 🃏 Blackjack (`04-blackjack`)
Jeu de Blackjack jouable (cartes aléatoires, somme, gain/perte).
**Concepts :** booléens, conditions, tableaux, boucles, `Math.random()`,
opérateurs logiques, objets et méthodes.

### 📝 Practice (`02-practice-part1`)
Petits exercices de logique (variables, types, conditions, calculs).

## ▶️ Comment lancer un projet
Ouvrir le fichier `index.html` du projet dans un navigateur.
